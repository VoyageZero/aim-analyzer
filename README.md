# Aim Analyzer
- Aim tracker for Kovaaks and AimLabs to track performance overtime.
- Currently under development.

## v0.1.0
- Planned features (Kovaaks Only)
  - Includes tracking for local files
  - Filter scenarios & dates.
  - Generate plots showcasing performance in different areas.

## Build From Source
### Dependencies
- [VCPKG](https://learn.microsoft.com/en-us/vcpkg/get_started/get-started?pivots=shell-bash)
  - Install dependencies from `vcpkg.json`
```bash 
vcpkg install
```
